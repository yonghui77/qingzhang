# 轻账 PWA（无需 Mac）

这是可以安装到 iPhone 主屏幕的网页 App。数据保存在当前 iPhone 的 Safari 本地存储中，支持离线使用。

## 部署

PWA 必须通过 HTTPS 网站访问。把本文件夹发布到任意静态网站服务即可，例如：

- GitHub Pages
- Cloudflare Pages
- Netlify
- Vercel

所有文件都是纯静态文件，不需要服务器程序或数据库。

### 使用 GitHub Pages 免费发布

1. 登录 GitHub，新建一个公开仓库，例如 `qingzhang`。
2. 把本文件夹内的所有文件上传到仓库根目录，确保 `index.html` 在最外层。
3. 打开仓库的 **Settings → Pages**。
4. 在发布来源中选择从分支发布，选择 `main` 分支和根目录。
5. 等待页面显示网站地址，通常形如 `https://用户名.github.io/qingzhang/`。
6. 用 iPhone Safari 打开这个地址，再添加到主屏幕。
7. 更改时间
发布到网站的只是程序代码，不包含你的账目；以后录入的账目仍只保存在手机本地。

## 安装到 iPhone

1. 用 iPhone 的 Safari 打开部署后的网站地址。
2. 点击 Safari 底部的“分享”按钮。
3. 向下找到并点击“添加到主屏幕”。
4. 名称保留为“轻账”，点击右上角“添加”。
5. 以后直接从桌面图标打开。

首次打开时需要联网加载一次；成功加载后可离线使用。

## 数据安全

- 数据只保存在安装该 PWA 的浏览器中。
- 清理 Safari 网站数据或更换手机可能导致数据丢失。
- 建议定期进入“设置 → 导出备份”，把 JSON 文件保存到 iCloud Drive。
- 恢复时选择之前导出的 JSON 文件即可。
